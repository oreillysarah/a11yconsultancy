When to use aria labels - coding tips for accessibility

It is better to use native html!! E.g.

input type="radio".. As this automatically focuses element. You can use aria for legacy code for custom controls, JS menus, certain JS events. There are times when the <label> element falls short—it cannot be used to provide multiple labels for a single form control, or to associate a single label with multiple form controls. There are also times when there is not a visible text label to be associated to a form control (although there should always be visible text where possible!). The majority of these labeling limitations can be overcome with three ARIA properties (aria-labelledby, aria-describedby, and aria-label). .
